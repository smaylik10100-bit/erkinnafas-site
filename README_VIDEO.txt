# Инструкция по видео
Поместите ваш файл ролика в папку `public` с именем **clinic-video.mp4**.
На сайте он доступен по адресу `/clinic-video.mp4` и проигрывается по кругу (autoplay, muted, loop, playsInline).

Порядок действий:
1) Скопируйте ваш MP4 в `public/clinic-video.mp4`
2) Локально проверьте: `npm install`, `npm run dev` → откройте http://localhost:5173
3) Закоммитьте и запушьте в GitHub → Vercel сам пересоберёт и обновит сайт.
